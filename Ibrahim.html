Hallo allemaal. Ik ga jullie uitleggen wat Visual Studio code is, wat het doet en waarom ik het gebruik.

Visual studio code is een programma waar in je een eigen website kan bouwen door middel van: Javascript, Html, Python enz. Ik gebruik HTML omdat ik het makkelijk vind. Het makkelijkste is om nadat je Visual Studio Code gelijk in het programma de extension: Live Server, te downloaden. Nadat je dat hebt gedaan kun je een eigen map maken waar in al je werk wordt opgeslagen. kies een taal waarin je wil programeren. Kies je HTML kun het best beginnen met een "!" te typen. klik op Emmet Abbreviation en je krijgt gelijk een plek om te beginnen!
